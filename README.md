# 🚀 YouTube Channel Analysis Tools

## 📋 **シンプルで使いやすい 3 つのツール**

### 🎯 **おすすめの使い方（シンプル順）**

#### 1️⃣ **YouTube 分析（メイン推奨）** ⭐️

**ファイル**: `4_channelCheck.gs` (7,736 行)

**🚀 基本操作:**

1. メニュー「YouTube 分析」→「⚙️ 初期設定」
2. B8 セル（入力欄）に `@YouTube` を入力
3. メニュー「YouTube 分析」→「🚀 チャンネル分析」

**⚡ シンプルメニュー構成:**

- **⚙️ 初期設定** - API キー・OAuth 認証を一括設定
- **🚀 チャンネル分析** - ワンクリックで完全分析実行
- **🔧 詳細分析** - 必要に応じて個別分析（サブメニュー）
- **❓ 使い方ガイド** - 困った時のヘルプ

---

#### 2️⃣ **YouTube ベンチマーク（複数比較）**

**ファイル**: `2_benchmark.gs` (2,354 行)

**🚀 基本操作:**

1. メニュー「📊 YouTube ベンチマーク」→「⚙️ API 設定」
2. 個別分析：「🔍 チャンネル分析」
3. 複数比較：データシートの B 列に@ハンドル名リスト →「📊 複数チャンネル取得」

**⚡ シンプルメニュー構成:**

- **⚙️ API 設定** - 初期設定
- **🔍 チャンネル分析** - 1 つのチャンネル詳細分析
- **📊 複数チャンネル取得** - リスト形式で一括取得
- **📈 レポート作成** - ベンチマークレポート作成

---

#### 3️⃣ **YouTube 事業分析（高度）**

**ファイル**: `youtube_business_intelligence.gs` (1,127 行)

**🚀 基本操作:**

1. メニュー「🚀 YouTube 事業分析」→「⚙️ 初期設定」
2. ダッシュボード B5 セルにチャンネル入力
3. 「🎯 チャンネル分析」で包括的事業分析

**⚡ シンプルメニュー構成:**

- **⚙️ 初期設定** - システム設定一括実行
- **🎯 チャンネル分析** - 包括的事業分析
- **🔧 詳細分析** - 専門分析（サブメニュー）

## ⚙️ **共通設定**

### API 設定

1. [Google Cloud Console](https://console.cloud.google.com/) でプロジェクト作成
2. YouTube Data API v3 を有効化
3. API キーを生成
4. 各システムの「API 設定」でキーを入力

### OAuth 認証（詳細分析のみ）

1. Google Cloud Console で OAuth 2.0 クライアント作成
2. 詳細分析システムで「🔑 OAuth 認証再設定」実行

## 🚀 使い方

### 1. 初期設定

1. Google Spreadsheets で新しいスプレッドシートを作成
2. 「拡張機能 > Apps Script」を開く
3. 上記ファイルをすべてコピー&ペースト
4. YouTube Data API v3 を有効化し、API キーを取得

### 2. 基本操作

1. **統合ダッシュボードの起動**

   - スプレッドシートを開くと自動で「📊 統合ダッシュボード」が表示
   - または「YouTube ツール > 🏠 統合ダッシュボード」から手動起動

2. **API 設定**

   - クイックアクセスの「⚡ API 設定」をクリック
   - 取得した YouTube Data API キーを入力

3. **チャンネル分析**
   - 「📊 既存チャンネル分析」: 個別チャンネルの詳細分析
   - 「🔍 ベンチマーク分析」: 複数チャンネルの比較分析

### 3. ベンチマーク分析の手順

1. データシートにハンドル名（@username）を入力
2. 「📈 チャンネル情報取得」で一括データ取得
3. 「📊 ベンチマークレポート」で分析レポート作成
